# Smart Agriculture System

AI-powered smart farming system with:
- Crop prediction
- Fertilizer recommendation
- Market price analysis
- Weather integration
- AI farming recommendations

# Setup

## Backend

```bash
cd backend
python -m venv .venv
.venv\Scripts\activate

pip install -r requirements.txt

python manage.py runserver
```

## Frontend

```bash
cd frontned

npm install

npm run dev
```

Frontend:
http://localhost:8080

Backend:
http://127.0.0.1:8000
